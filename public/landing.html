FAQ

Common questions

Do I need an API key to use NOVA?
No — NOVA includes shared API keys for all providers so you can start immediately. If you want higher limits or to use your own quota, you can add your own key in Settings → BYOK Keys.
Is my data private?
Your conversations are stored in a private database only you and the admin can access. We don't sell data or use your conversations to train models. You can export or delete your data at any time from Settings.
What's the difference between Chat and Agent mode?
Chat is a standard back-and-forth conversation. Agent mode lets NOVA work autonomously — it breaks your task into steps, searches the web for information, saves findings, and delivers a complete result without you needing to guide every step.
Can I use NOVA with local models?
Yes. NOVA supports Ollama out of the box. Install Ollama, run ollama serve, and select Ollama as your provider. All processing stays on your machine.
What file types can I attach?
PDFs, plain text, Markdown, code files (JS, TS, Python, JSON, CSV, HTML, CSS), and more. Max 5MB per file, up to 5 files per message. NOVA extracts and reads the text content to answer your questions.
